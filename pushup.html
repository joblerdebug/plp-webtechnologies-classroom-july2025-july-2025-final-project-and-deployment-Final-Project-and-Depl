Usage Guidelines

Logo Clear Space

Maintain clear space around the logo equal to the height of the 'A' in Afribio on all sides.

Minimum Size

Never use the logo smaller than 24mm in width for print or 48px for digital applications.

Color Combinations

Use primary green (#709a36) as the main brand color. Gold (#ffce08) should be used as an accent color.

Typography Hierarchy

Use Inter Bold for headings, Inter SemiBold for subheadings, and Inter Regular for body text.
